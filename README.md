# Library Application

## 📌 Description
A web-based application for managing a collection of books, allowing users to add, remove, and track reading status.

## 🚀 Features
- Add and remove books
- Track reading status
- Dynamic content rendering
- User-friendly interface

## 🛠️ Tech Stack
- JavaScript
- HTML
- CSS

## 📚 What I Learned
- Object-oriented programming in JavaScript
- Managing collections of data
- Dynamic UI updates
- User input handling
